习题 0

  1. 令 A, B, C 分别为如下三集合 (其中 `RR` 为所有实数的集合):

    `{x in RR | -1 le x le 1}`,

    `{x in RR | -2 lt x lt 1}`,

    `{x in RR | x lt 0}`.

    计算 A `nn` (B `uu` C), A `uu` (B `nn` C).
  2. 证明: 若集合有包含关系 M `sube` N, 则 M `nn` N = M, M `uu` N = N.
  3. 证明: 对于任意集合 M, N, L, 有
    `M nn (N uu L) = (M nn N) uu (M nn L)`.
  4. 令 C 为一集合. 证明: 当 A, B `sube` C 时,
    `[(A-B) uu (B-A)]^C = (A-B)^C nn (B-A)^C`,
    其中 X `sube` C 时, `X^C` 记为 `bar X` (X 在 C 中的补集).
  5. 以 A = {1, 2, 3} 上的变换为例, 说明映射的合成不满足交换律.
  6. 指出下列映射的单射, 满射和双射性, 并求出每一双射的逆映射:
    1. 令 `a in RR`, `a gt 0`. 映射
      `f: RR rarr RR^+` (正实数全体), `x |-> a^x`.
      将 `RR^+` 改为 `RR` 呢?
    2. 映射
      `g: RR^+ rarr RR^+`, `x |-> 1/x`;
    3. 映射
      `h: RR rarr RR`, `x |-> { ( x, x lt 0; ), ( 1, 0 le x lt 1; ), ( 2x-1, x ge 1. ) :}`
    4. 映射
      `k: RR rarr RR`, `x |-> { ( x, x lt 0; ), ( 0, 0 le x lt 1; ), ( x-1, x ge 1. ) :}`
  7. 给定两个映射 `f: A rarr B`, `g: B rarr C`, 令 `h = g @ f`. 证明:
    1. 若 `h` 是一单射 (满射), 则 `f(g)` 也是一单射 (满射). 反之是否正确?
    2. 若 `f`, `g` 都是双射, 则 `h` 也是双射, 且
      `h^(-1) = f^(-1) @ g^(-1)`.
  8. 用数学归纳法证明:
    1. `1 * 1! + 2 * 2! + cdots + n * n!` `= (n+1)!-1`;
    2. 令 h 为一正整数, 则对于所有正整数 n, 有 `(1+h)^n ge 1 + nh`;
    3. 令 `n, r in ZZ`, `n ge 1`, `r ge 0`, 且 `n ge r`. 我们从已知 n 个元素中取 r 个的组合数为
      ` C_n^r = ((n),(r)) = (n!)/(r!(n-r)!) = [n(n-1) cdots (n-r+1)]/(r!)`,
      我们又令 `0! = 1`, `((n),(0)) = 1`. 则
      `((n),(r-1)) + ((n),(r)) = ((n+1),(r))`;
    4. 二项式定理
      `(x+y)^n = sum_(i=0)^n ((n),(i)) x^(n-i) y^i`,
      即
      ` (x+y)^n = x^n + ((n),(1)) x^(n-1) y + cdots + ((n),(r)) x^(n-r) y^r + cdots + y^n`,
      并由此证明: 含 n 个元素的集合的子集恰有 `2^n` 个.
  9. 令 `a, b in ZZ`, 且不全为零. 证明: 若 `a = da_1`, `b = db_1`, 则 `(a,b) = |d| (a_1, b_1)`.
  10. 令 `p_i` 为素数, `i = 1, 2, cdots n`, 且两两不同, 又令 `a = 1 + prod_(i=1)^n p_i` (即 `a =1 + p_1 p_2 cdots p_n`). 证明: `p_i` 不整除 `a`, `i = 1, 2, cdots n`, 进而证明: 素数是无限多的.
  11. 证明: 若 m 和 n 互素, 则 `2^m - 1` 和 `2^n - 1` 也互素.
  12. 判断下列数集是否为数域:
    1. `QQ[i] = {a + bi | a, b in QQ, i = sqrt(-1)}`;
    2. `S = {1/(2^n) | n in ZZ}`;
    3. ` ZZ(pi) = {(a_n pi^n + a_(n-1) pi^(n-1) + cdots + a_1 pi + a_0) / (b_m pi^m + b_(m-1) pi^(m-1) + cdots b_1 pi + b_0) | n, m in ZZ, n, m, ge 0`, `a_i, b_j in ZZ, i = 1, 2, cdots, n, j = 1, 2, cdots, m,` 且 `b_m, b_(m-1), cdots, b_1, b_0` 不全为零`}`.
  13. 求 `k, l, m`, 使得
    `(2x^2 + lx - 1)(x^2 - kx + 1) = 2x^4 + 5x^3 + mx^2 - x - 1`.
  14. 举例说明下列情况都可能出现:
    1. `del(f(x) + g(x)) lt max{del f(x), del g(x)}`;
    2. `del(f(x) + g(x)) = max{del f(x), del g(x)}`;
    3. 将 (1) (2) 中的 max 换为 min.
  15. 令 `f(x), g(x), h(x) in RR[x]`. 证明: 若
    `f^2(x) = xg^2(x) + xh^2(x)`,
    则 `f(x) = g(x) = h(x) = 0`. 用 `CC` 代替 `RR` 时, 上述事实还成立 吗? 为什么?
  16. 证明:
    ` 1 - x + [x(x-1)]/2 + cdots + (-1)^n [x(x-1)cdots(x-n+1)]/(n!)` `= (-1)^n [(x-1)cdots(x-n)]/(n!)`.
  17. 由等式 `(1+x)^m (1+x)^n = (1+x)^(m+n)` 证明下式:
    ` C_m^k + C_m^(k-1) C_n^1 + cdots + C_m^1 C_n^(k-1) + C_n^k = C_(m+n)^k`,
    其中 m, n 为正整数, k = 0, 1, ..., m+n. 此处约定, 对于正整数 s, t, 当 s > t 时, `C_t^s = 0`.
  18. 令 `f(x), g(x), q(x), r(x) in bbb"P"[x]`. 证明: 若
    `f(x) = q(x) g(x) + r(x)`,
    且
    `del r(x) lt del g(x) le del f(x)`,
    则
    `del q(x) = del f(x) - del g(x)`.
  19. 令 `f_i(x), g_i(x) in bbb"P"[x]`, `i = 1, 2`. 证明: 若 `f_1(x) != 0`, 且
    `g_1(x) g_2(x) | f_1(x) f_2(x)`, `f_1(x) | g_1(x)`,
    则
    `g_2(x) | f_2(x)`.
  20. 令 `f(x), g_1(x), g_2(x) in bbb"P"[x]`. 证明:
    `(g_1(x) - g_2(x) | (f(g_1(x)) - f(g_2(x)))`.
  21. 证明: 当且仅当 m, p, q 适合什么条件时, 有 `g(x) | f(x)`:
    1. `f(x) = x^3 + px + q`, `g(x) = x^2 + mx - 1`;
    2. `f(x) = x^4 + px^2 + q`, `g(x) = x^2 + mx + 1`.
  22. 求 `g(x)` 除 `f(x)` 的商式和余式:
    1. `f(x) = 2x^5 - 5x^3 - 8x`, `g(x) = x + 3`
