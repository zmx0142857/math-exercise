15. 设 f(x) 是 [0, 1] 上的连续函数, f(0) = 0, 且 `f(x)/x` 在 [0, 1] 上可积. 证明: `lim_(n to oo) n int_0^1 f(x^n) dx = int_0^1 f(x)/x dx`.

    作变元变换 `t = x^n`, 则左端积分等于 `n int_0^1 f(t) d(t^(1/n)) = int_0^1 f(t)/t t^(1/n) dt`. 因为 `x^(1/n)` 在 [0, 1] 上不一致收敛, 我们把区间分为两部分来讨论. `{: (, |n int_0^1 f(x^n) dx - int_0^1 f(x)/x dx| ),( =, |int_0^1 f(x)/x (x^(1/n) - 1) dx| ),( le, int_0^delta |f(x)/x| |x^(1/n) - 1| dx + int_delta^1 |f(x)/x| |x^(1/n) - 1| dx. ):}` 我们记 M 为 `|f(x)/x|` 在 [0, 1] 上的上界. 对任意 `epsilon gt 0`, 先取 `delta` 充分小, 如 `delta = epsilon/(2M)`, 使第一项小于 `epsilon/2`. 取定此 `delta` 后, 由 `lim_(n to oo) delta^(1/n) = 1` 知 `EE N in NN`, 当 n > N 时有 `|x^(1/n) - 1| le |delta^(1/n) - 1| lt epsilon/(2M)`, `AA x in [delta, 1]`. 从而第二项小于 `(1-delta) M epsilon/(2M) lt epsilon/2`. 综上可知所要证的极限成立.
