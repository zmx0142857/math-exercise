1. 应用第一换元积分法, 求以下不定积分:
    1. `int root 3 (1-2x) dx`;
    2. `int dx/(3+2x^2)`;
    3. `int dx/sqrt(3-2x^2)`;
    4. `int (x dx)/sqrt(1+x^2)`;
    5. `int (x dx)/(3+2x^2)`;
    6. `int (x dx)/(1+x^4)`;
    7. `int dx/(sqrt x (1+x))`;
    8. `int dx/(x^2 "e"^(1/x))`;
    9. `int dx/(x sqrt(x^2+1))`;
    10. `int dx/(x^2+1)^(3/2)`;
    11. `int dx/sqrt(x(1-x))`;
    12. `int (x dx)/sqrt(1+x)`.
    1. 令 `t = 1-2x`, 原式 = ` -1/2 int t^(1/3) dt = -3/8 t^(4/3) + C = -3/8 (1-2x)^(4/3) + C`;
    2. 令 `t = sqrt(2/3) x`, 原式 = ` 1/3 int (sqrt(3/2) dt)/(1+t^2) = 1/sqrt 6 arctan t + C = sqrt 6/6 arctan {:sqrt 6/3 x:} + C`;
    3. 令 `t = sqrt(2/3) x`, 原式 = ` 1/sqrt 3 int (sqrt (3/2) dt)/sqrt(1-t^2)` `= 1/sqrt 2 arcsin t + C` `= sqrt 2/2 arcsin {:sqrt 6/3 x:} + C`;
    4. 令 `t = sqrt(1+x^2)`, 则 `x^2 = t^2-1`, `x dx = t dt`, 原式 = `int (t dt)/t = t + C` `= sqrt(1+x^2) + C`;
    5. 令 `t = 3 + 2x^2`, 则 `x^2/2 = (t-3)/4`, `x dx = dt/4`,原式 = ` int dt/(4t) = 1/4 ln t + C` `= 1/4 ln(3+2x^2) + C`;
    6. 令 `t = x^2`, 原式 = `1/2 int dt/(1+t^2) = 1/2 arctan t + C = 1/2 arctan x^2 + C`;
    7. 令 `t = sqrt x`, 则 `dx = 2t dt` 原式 = `int (2 dt)/(1+t^2) = 2 arctan t + C` `= 2 arctan sqrt x + C`;
    8. 令 `t = 1/x`, 则 `dx = -dt/t^2`, 原式 = `-int dt/"e"^t = "e"^-t + C = "e"^(-1/x) + C`;
    9. 令 `t = sqrt(x^2+1)`, 则 `x dx = t dt`, 原式 = ` int dt/(t^2-1)` `= 1/2 int (1/(t-1) - 1/(t+1)) dt` `= 1/2 ln |(t-1)/(t+1)| + C` `= 1/2 ln |(sqrt(x^2+1)-1)/(sqrt(x^2+1)+1)|+C`;
    10. 令 `t = 1/sqrt(x^2+1)`, 则 `dt = -(x dx)/(x^2+1)^(3/2)`, `x^2 = 1/t^2 - 1`, 原式 = ` -"sgn"x * int dt/sqrt(1/t^2-1)` `= "sgn"x* sqrt(1-t^2) + C` `= ("sgn"x *|x|)/sqrt(x^2+1) + C` `= x/sqrt(x^2+1) + C`;
    11. 令 `t = sqrt(1-x)`, 则 `x = 1-t^2`, 原式 = ` int (-2t dt)/(t sqrt(1-t^2))` `= -2 arcsin t + C` `= -2 arcsin sqrt(1-x) + C`;
    12. 令 `t = sqrt(1+x)`, 则 `x = t^2-1`, 原式 = ` int ((t^2-1) 2t dt)/t` `= 2t(t^2/3-1) + C` `= 2/3 (x-2) sqrt(1+x) + C`.

  3. 应用第一换元积分法, 求以下不定积分 (再续):

    3. `a^2 = b^2 = 0` 时, 原式 = `int x^-4 dx = -1/(3x^3) + C`; `a^2 = b^2 != 0` 时, 记原式 = `I`, 应用分部积分, ` int dx/(a^2 + x^2)` `= x/(a^2+x^2) + 2 int (x^2+a^2-a^2)/(a^2+x^2)^2 dx` `= x/(a^2+x^2) + 2 int dx/(a^2+x^2) - 2a^2 I`, 所以 `I = 1/(2a^2) (x/(a^2+x^2) + 1/a arctan {:x/a:})`; `a^2 != b^2` 时, 原式 = ` 1/(a^2-b^2) int (1/(b^2+x^2) - 1/(a^2+x^2)) dx`, 其中 `int dx/(a^2+x^2) = { (1/a arctan{:x/a:}+C),(-1/x+C):}`;
    4. 原式 = `1/2 int ("d"(x^2))/((x^2+3/2)^2 - 1/4) = 1/2 ln {:(x^2+2)/(x^2+1):} + C`;

    11. `int sqrt(a^2-x^2)/x^2 dx`;
    12. `int sqrt(x^2-a^2)/x^2 dx`.
    1. `x = a sin t`, 原式 = ` int (a cos t dt)/(a cos t)^3 = 1/a^2 tan t + C = x/(a^2 sqrt(a^2-x^2)) + C`;
    2. `x = a sin t`, 原式 = ` int (a^2(1-cos^2 t) dt)/(a^2 cos^2 t)` `= int (1/(cos^2 t) - 1) dt` `= tan t - t + C` `= x/sqrt(a^2-x^2) - arcsin{:x/a:} + C`;
    3. `x = a sin t`, 原式 = ` int a^2 sin^2 t dt` `= a^2/4 int(1-cos 2t) "d"(2t)` `= a^2/4 (2t - sin 2t) + C` `= a^2/2 arcsin {:x/a:} - x/2 sqrt(a^2-x^2) + C`;

    5. `x = a cosh t`, 原式 = ` int a^2 cosh^t t dt` `= a^2/4 int(cosh 2t +1) "d"(2t)` `= a^2/4 (sinh 2t + 2t) + C'` `= x/2 sqrt(x^2-a^2) - a^2/2 ln(x+sqrt(x^2-a^2)) + C`;
    6. 同 (1)
    7. `x = a cosh t`, 原式 = ` int a^2 sinh^2 t dt` `= a^2/4 int (cosh 2t - 1)"d"(2t)` `= a^2/4 (sinh 2t - 2t) + C'` `= x/2 sqrt(x^2-a^2) + a^2/2 ln(x+sqrt(x^2-a^2)) + C`;
