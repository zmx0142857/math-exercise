1. 根据导数的定义, 直接求下列函数的导数:
    1. `x^3 + 2x`;
    2. `x/(x-1)`;
    3. `root 3 (x^2)`;
    4. `tan x`;
    5. `cot x`.
    1. ` (x^3 + 2x)'` `= lim_(Delta x to 0) ((x+Delta x)^3 + 2(x + Delta x) - x^3 - 2x) / (Delta x)` `= lim_(Delta x to 0) (3x^2 + 3x Delta x + (Delta x)^2 + 2)` `= 3x^2 + 2`.
    2. ` (x/(x-1))'` `= lim_(Delta x to 0) ((x+Delta x)/(x + Delta x - 1) - x/(x-1)) / (Delta x)` `= lim_(Delta x to 0) -1/((x-1)(x+Delta x-1))` `= -1/(x-1)^2`.
    3. ` (root 3 (x^2))'` `= lim_(Delta x to 0) (root 3 ((x+Delta x)^2) - root 3 (x^2)) / (Delta x)` `= lim_(Delta x to 0) ((x+Delta x)^2 - x^2) / (Delta x ( root 3 ((x+Delta x)^4) + root 3 ((x + Delta x)^2 x^2) + root 3 (x^4) ))` `= lim_(Delta x to 0) (2x + Delta x) / ( root 3 ((x+Delta x)^4) + root 3 ((x + Delta x)^2 x^2) + root 3 (x^4) )` `= 2/(3 root 3 x)`.
    4. ` (tan x)'` `= lim_(Delta x to 0) (tan(x+Delta x) - tan x) / (Delta x)` `= lim_(Delta x to 0) (tan Delta x (1 + tan(x+Delta x) tan x)) / (Delta x)` `= 1 + tan^2 x`.
    5. ` (cot x)'` `= lim_(Delta x to 0) (cot(x+Delta x) - cot x) / (Delta x)` `= lim_(Delta x to 0) (cot x cot(x+Delta x) + 1) / (cot (-Delta x) Delta x)` `= -1 - cot^2 x`.
